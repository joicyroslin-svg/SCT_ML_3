# SCT_ML_3 - Bank Marketing Prediction using Decision Tree

## 📌 Project Overview
This project uses a Decision Tree Classifier to predict whether a customer will purchase or subscribe to a product/service based on demographic and behavioral data.

## 🎯 Objective
To build a classification model that predicts customer response using the Bank Marketing Dataset.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 🤖 Algorithm Used
Decision Tree Classifier

## 📊 Dataset
Bank Marketing Dataset

## 🚀 Project Workflow
1. Loaded the dataset
2. Checked dataset information and missing values
3. Encoded categorical columns
4. Split data into training and testing sets
5. Trained a Decision Tree Classifier
6. Evaluated the model using accuracy and classification report
7. Visualized confusion matrix, decision tree, and feature importance

## 📈 Output
- Model Accuracy
- Classification Report
- Confusion Matrix
- Decision Tree Visualization
- Feature Importance Graph

## ✅ Result
The Decision Tree model successfully predicts whether a customer is likely to purchase/subscribe based on given features.

- Data Visualization
